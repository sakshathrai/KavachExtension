FAQS

Have questions? Here you will find answers most of the times, along with step-by-step guide.

What is browser extension?
A browser extension is a small software application that adds a capacity or functionality to a web browser.
What does KAVACH extension do?
KAVACH extension detects dark patterns on web page and also classifies the type of dark pattern.
How can we use KAVACH extension?
To use KAVACH extension, you will have to select the KAVACH extension folder then click on it.
Does this extension access our personal information?
The answer is No. KAVACH extension does not require or access any of the personal data of users and it has compilance with GDP and CCPA.
Can users customize the extension?
The extension is customizable, users can customize it accordingly.
Does extension work automatically or manually?
The extension works both automatically and manually. Its completely based on users choice to use any option.
